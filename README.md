# Signup Bot
A Discord bot that handles signups within a Discord server and pulls from r/memes.

## Setting up the virtual environment
### Windows
```
python -m venv .
cd Scripts
activate
pip install -r requirements.txt
```

### macOS/Linux
```
python3 -m venv env
source env/bin/activate
pip3 install -r requirements.txt
```
